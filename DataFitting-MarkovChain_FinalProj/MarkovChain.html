a) Xy dng ma trn chuyn trng thi P ca xch Markov

T d liu cho, ta xy dng c vector A v B ln lt l s phng hc cc trng thi trong nm 2020 v 2021: $$ A = [34, 12, 3, 1] $$ $$ B = [26, 17, 5, 2] $$

V ma trn X: $$ X = \begin{bmatrix} x_1 & x_2 & x_3 & x_4 \cr y_1 & y_2 & y_3 & y_4 = 0 \end{bmatrix}$$

Vi:

  • $x_i$ l s phng hc gi nguyn trng thi $i$.
  • $y_i$ l s phng hc chuyn t trng thi $i$ sang trng thi $(i + 1)$.
  • $i \in (1, 4)$

T ta c c biu thc:

  • $ x_i = B_i - A_{i - 1} + x_{i - 1} $ (1)
  • $ y_i = A_i - x_i $ (2)

Khi , ta tnh c ma trn chuyn trng thi P ca xch Markov: $$ P = \begin{bmatrix} \frac{x_1}{34} & 0 & 0 & 0 \cr \frac{y_1}{34} & \frac{x_2}{12} & 0 & 0 \cr 0 & \frac{y_2}{12} & \frac{x_3}{3} & 0 \cr 0 & 0 & \frac{y_3}{3} & \frac{x_4}{1} \end{bmatrix}$$

state_2020 = [34, 12, 3, 1]

def find_X(A, B):
    ''' 
        Tm ma trn X, size 2x4, vi dng u l x_i, dng sau l y_i.
        A: vector trng thi cc phng hc nm 2020
        B: vector trng thi cc phng hc nm 2021
        Return: ma trn X size 2x4
    '''
    n = len(A)
    X = [[0 for _ in range(n)], [0 for _ in range(n)]]
    
    X[0][0] = 26 # c 24 phng hc gi nguyn trng thi 1 
    X[0][3] = 1 # c 1 phng hc gi nguyn trng thi 4

    for i in range(1, n):
        X[0][i] = B[i] - A[i - 1] + X[0][i-1] # x_i = B_i - A_(i-1) + X_(i-1)
        X[1][i-1] = A[i-1] - X[0][i-1] # y_i = A_i - x_i

    return X

def multiply_matrix_matrix(A, B):
    ''' 
        Nhn hai ma trn
        A: matrix size nxm
        B: matrix size mxp
        return: matrix size nxp
    '''
    # Kim tra xem s ct ca A c bng s hng ca B khng
    if len(A[0]) != len(B):
        raise ValueError("S ct ca A phi bng s hng ca B  nhn ma trn.")
    
    result = [[0 for _ in range(len(B[0]))] for _ in range(len(A))]
    
    for i in range(len(A)):
        for j in range(len(B[0])):
            for k in range(len(B)):
                result[i][j] += A[i][k] * B[k][j]
    
    return result

def matrix_pow(A, power):
    '''
        calculate the nth power of matrix A using exponentiation by squaring
        A: ma trn size nxn
        power: s m
    '''

    # to ma trn n v size nxn
    result = [[1 if i == j else 0 for j in range(len(A))] for i in range(len(A))]
    base = A

    while power > 0:
        if power % 2 == 1:
            result = multiply_matrix_matrix(result, base)
        base = multiply_matrix_matrix(base, base)
        power //= 2
    
    return result
